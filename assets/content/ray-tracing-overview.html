Ray Tracing in C/C++

I built a CPU-based ray tracer in C/C++ using backward ray tracing from a fixed camera. The core implementation includes sphere and triangle intersections, per-light shadow rays, and Phong illumination with proper clamping. For triangle shading, I used barycentric interpolation for normals and material properties to produce smooth lighting and highlights.

Beyond the core renderer, I implemented recursive reflection, soft shadows, anti-aliasing, and animation output. I tested these features across multiple scenes to compare baseline rendering against enhanced visual quality.
