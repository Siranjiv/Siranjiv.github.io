2D Simulator for Mass spring, Linear FEM and Corotational FEM

I developed a 2D deformable object simulation system using C++ and OpenGL, implementing mass-spring, linear FEM, and corotational FEM models. Built on a provided framework with rendering and GUI support (ImGUI), the project enables real-time deformation of 2D mesh objects under external forces and constraints.

The system computes physical behavior through mass matrix construction, elastic energy, internal forces, and stiffness matrices. I implemented core algorithms for each model: Hooke’s law for the mass-spring system, linear elasticity for FEM, and rotation-invariant deformation using polar decomposition for corotational FEM. These approaches allow accurate simulation of both small and large deformations.

Additional features include interactive vertex manipulation, configurable simulation parameters, multiple demo meshes, and visualization of fixed and dynamic vertices. The system supports switching between force models to compare their behavior and deformation quality in real time.

This project strengthened my understanding of physics-based simulation, numerical methods for deformation, and the mathematical foundations of FEM, including matrix operations and rotational invariance in dynamic systems.
