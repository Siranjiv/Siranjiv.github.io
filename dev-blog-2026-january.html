Cemented Dreams

January 2026 Dev Blog

Siranjiv Sivapalan | Weekly Engineering Notes

Development resumed in week 3 after semester break. This month focused on introducing debris VFX for the Grow Transformation system, then finalizing the feature with integration planning for the broader production pipeline.

Note: Following the semester break, development resumed in week 3. No tasks were assigned during weeks 1 and 2.

January Snapshot

  • Researched and learned Niagara fundamentals to support a new VFX feature request.
  • Prototyped debris VFX for fractured Grow platforms during initial implementation.
  • Completed the debris VFX system and validated behavior with game-direction review.
  • Documented integration constraints and finalized a practical rollout plan with designers.

Week 3

Debris VFX Foundation

This week I was assigned a new look-and-feel feature for the Grow Transformation system: debris VFX on each fractured platform. The objective was to improve visual feedback and make transformations feel more dynamic and impactful.

Because this work required Unreal Engine Niagara and I had not used Niagara previously, I started by learning its core workflow while building an initial base implementation in parallel. The result is an early but functional VFX foundation that will support future polish and customization.

Week 4

Debris VFX Completion

This week I completed debris VFX integration for the Grow Transformation system, applying effects to each fractured platform to improve overall visual impact.

  • Implemented using Blueprint instances attached to fractured platforms.
  • Identified a cleanup limitation: removing the Grow system does not automatically remove associated Blueprint instances.
  • Presented the implementation and limitation to the game director; solution was approved as a separate system for now.
  • Planned integration timing after designers complete level design, enabling smoother final rollout.

Overall, this week focused on shipping the feature and documenting the practical considerations needed for stable pipeline integration.
