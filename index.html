Jello Cube Simulation

I developed a physics-based simulation of a Jello Cube using a mass-spring system in C++ with OpenGL for visualization. The simulation models an 8×8×8 grid of mass points connected by structural, shear, and bend springs, and incorporates damping and external force fields. I implemented both explicit Euler and Runge-Kutta 4th order (RK4) integration methods to explore numerical stability in dynamic systems.

This project taught me how to simulate soft-body dynamics using Newtonian physics and Hooke’s Law, as well as how to handle real-time collisions using a spring penalty approach. I also added extra features such as collision with an internal cube and rendered the simulation within a Cornell Box environment for better visual clarity. Through this work, I gained a deeper understanding of numerical integrators, collision response, and the challenges of real-time physical simulation in animation.

Motion Capture Interpolation

I developed a motion capture interpolation system in C++ and OpenGL that processes skeleton data (ASF) and motion data (AMC) to generate smooth intermediate animations. The system supports four interpolation methods: Linear Euler, Bezier Euler (via de Casteljau’s algorithm), Linear Quaternion (SLERP), and Bezier Quaternion. The results are saved to a new AMC file and visualized in an OpenGL-based mocap player, with original motion shown in red and interpolated motion in green.

This project deepened my understanding of animation interpolation techniques, the differences between Euler angles and quaternions, and the implementation of real-time skeletal visualization. It strengthened my grasp of key concepts in motion data processing and animation systems.

Inverse Kinematics with Skinning

I developed an inverse kinematics (IK) system integrated with mesh skinning using C++ and OpenGL. Built on a forward kinematics (FK) foundation, the project enables real-time animation of a deformable character mesh by manipulating joint transformations. It supports Damped Least Squares (DLS) IK and Linear Blend Skinning (LBS) as core features.

For added flexibility, I implemented multiple IK solvers (DLS, Pseudoinverse, and DLS_ALT) with recursive subdivision for smoother handle movements. The system also supports switching between Linear Blend and Dual Quaternion Skinning to compare deformation results. Additional features include interactive keyboard controls and frame-by-frame screenshot capture for recording animations.

This project deepened my understanding of real-time character animation, skeletal deformation techniques, and matrix/quaternion transformations providing hands-on experience with the mathematical and visual aspects of animation systems.
