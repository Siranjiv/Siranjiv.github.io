About Me

I’m Siranjiv Sivapalan, a software engineer and game developer based in Los Angeles, currently pursuing my Master’s in Computer Science (Game Development) at the University of Southern California. Prior to graduate school, I worked as a software engineer focusing on full-stack development using the ASP.NET framework building and maintaining end-to-end web applications and Windows applications in WinForms.

With a strong foundation in programming and a passion for games and interactive media, I enjoy creating systems that blend logic, creativity, and user experience. My portfolio spans gameplay mechanics, animation programming, and data-driven visualizations, including graph-based projects. Whether working independently or on a team, I take pride in writing clean, efficient code and continuously growing as a developer.
